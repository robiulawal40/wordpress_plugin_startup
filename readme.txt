Missing woocommerce checkout field editor: a best and simple way to edit checkout fields.

== Description ==

A plugin based on woocommerce functionality. Using this plugin user will get a felling of woocommerce build-in UI.

<h4>Plugin Feature</h4>
<ul>
<li> <strong>Plugin Size:</strong> We tried to make the plugin less sized, currently 30kb  </li>
<li>Following Field types are available
<ul>
<li>Text Input </li>
<li>Password</li>
<li>Email</li>
<li>Phone</li>
<li>Select</li>
<li>Textarea</li>
<li>Radio</li>
</ul>
</li> 

</ul>

<h4>Why This plugin is different</h4>

1. No Extra Code
2. Easy to Edit 

== Installation ==

1. From the Dashboard:
2. Plugins
3. Add New
4. Install Now
6. Activate
7. Go to Dashboard:-> WooCommerce -> missing checkout form editor
